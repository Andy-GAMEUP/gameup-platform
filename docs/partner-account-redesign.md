# 파트너 계정 기능 재설계 요구사항

## 기업회원 계정 구분: 개발자 vs 파트너(게임서비스기업)

### 1. 개발자 계정 - 로그인 후 이동
- 로그인 이후 **개발사 콘솔(Developer Console)**로 이동

### 2. 개발자 센터 - 미니홈 관리 재설계
- 기존 콘솔 형태 유지
- **미니홈 관리 메뉴**: 기존 양식 삭제 → Figma partners/1 양식으로 변경
- 미니홈 생성 및 관리 기능 포함

### 3. 개발자 콘솔 - 게임 플레이어 피드백
- 기존 피드백 메뉴 기능과 페이지 유지 (변경 없음)

### 4. 개발자 콘솔 - 파트너사 메시지 관리
- 미니홈 메뉴 내에 추가
- 기업회원(파트너사)와 주고받는 메시지 관리:
  - 댓글, 신청 메시지 수신
  - 답변 기능
  - 메시지 관리 (보관/삭제 등)

### 5. 개발자 콘솔 → 서비스 사이트 네비게이션
- 서비스 사이트 홈으로 이동 시 상단 네비게이션:
  - 베타게임, 라이브게임, 커뮤니티, **파트너라운지(파트너 홈)**

### 6. 파트너 계정 - 콘솔 삭제, 홈 이동
- 기존 파트너 콘솔(사이드바 레이아웃) **삭제**
- 로그인 후 **서비스 홈 페이지**로 바로 이동
- 사이드바 없음, Figma 웹 페이지 디자인 참고

### 7. 파트너 계정 - 마이페이지/프로필 편집
- 마이페이지 및 설정: 기존 플레이어 양식 동일 유지
- **프로필 편집**: 기존 양식 삭제 → Figma partners/1 양식으로 변경
- 프로필 생성 및 관리 가능

### 8. 메시지 관리 기능 추가
- **파트너 계정 + 플레이어 계정** 모두 적용
- 받은 메시지 확인
- 답신 기능
- 보관/삭제 등의 관리 기능

### 9. 마이페이지 네비게이션 오류 수정
- 파트너 계정, 플레이어 계정 모두
- 마이페이지 이동 후 홈이나 다른 페이지로 이동되지 않는 버그 수정
